
### 添加超时返回逻辑
- 用`TextControl`举例：
```python
import uiautomation as ui
import time

class AiAutomation(object):

	@staticmethod
	def TextControl(parent: ui.WindowControl, Name=None, SubName=None, timeout=3):
		"""
		xxx
		xxx
		"""
		start_time = time.time()
		if Name is not None:
			while True:
				textControl = parent.TextControl(Name=Name)
				if textControl.Exists(timeout)
					return textControl
				elif (time.time() - start_time) > timeout:
					return None
				else:
					time.sleep(0.5)
		if SubName is not None:
			while True:
				textControl = parent.TextControl(SubName=SubName)
				if textControl.Exists(timeout)
					return textControl
				elif (time.time() - start_time) > timeout:
					return None
				else:
					time.sleep(0.5)
```
